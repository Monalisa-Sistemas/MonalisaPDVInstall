Instalador do MonalisaPDV

Primeira instalação do PDV no computador do cliente

Baixar instalador
Windows 64 bits · monalisapdv_install.exe · sempre a versão mais recente

Como instalar

  1. Baixe o instalador
    Clique no botão acima e salve o arquivo monalisapdv_install.exe.
  2. Execute como Administrador obrigatório
    Clique com o botão direito no arquivo → Executar como administrador.
  3. Siga o assistente de instalação
    Ele instala os componentes necessários (Visual C++, ACBr, TEF e drivers). Se aparecer, marque a instalação do driver do pinpad Gertec.
  4. Instale o PDV
    Ao final abre a janela do Monalisa Downloader. Clique em “Instalar PDV” — ele baixa a versão mais recente e instala em C:\MonalisaPDV.
  5. Inicie o sistema
    Clique em “Iniciar PDV” para abrir o MonalisaPDV.
Importante: é necessário conexão com a internet na primeira instalação — é quando o PDV é baixado. Se o Windows exibir um aviso de segurança (SmartScreen), clique em “Mais informações” → “Executar assim mesmo”.
