# MonalisaPDVInstall

Página de download do **instalador de primeira instalação** do MonalisaPDV, para uso do time de suporte.

## Página (GitHub Pages)

**https://monalisa-sistemas.github.io/MonalisaPDVInstall/**

Abra o link, clique em **"Baixar instalador"** e siga o passo a passo. O botão sempre baixa a versão mais recente.

## Link direto do instalador

https://github.com/Monalisa-Sistemas/MonalisaReleases/releases/download/pdv_installer/monalisapdv_install.exe

O instalador é gerado e publicado automaticamente pelo workflow `installer-windows.yml` do repositório `MonalisaPdv`.
